Servicios

Nuestros talentos a tu servicio.

FrontEnd

Desarrollo web que se dedica a la parte frontal de un sitio web, en pocas palabras del diseño de un sitio web, desde la estructura del sitio hasta los estilos como colores, fondos, tamaños hasta llegar a las animaciones y efectos.

Diseño Responsivo

“Responsivo” (adaptable). Esto significa hacer que un sitio web sea accesible y adaptable en todos los dispositivos: computadores, tabletas, smartphones, etc.

e-learning

Implementación de Plataformas educativas web o Entornos Virtuales de Enseñanza y Aprendizaje, es una aplicación web que integra un conjunto de herramientas para la enseñanza-aprendizaje en línea, permitiendo una enseñanza no presencial.
